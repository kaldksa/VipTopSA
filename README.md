# VipTopSA - Instagram Management Platform

## 📱 نظام إدارة حسابات Instagram احترافي

منصة متقدمة لإدارة حسابات Instagram مع ميزات متعددة مثل جدولة المنشورات، التحليلات، الرد الآلي على التعليقات، وإدارة الرسائل المباشرة.

---

## 🎯 الميزات

✅ **جدولة المنشورات** - جدول منشوراتك مسبقاً
✅ **تحليل الإحصائيات** - تقارير شاملة عن الأداء
✅ **الرد الآلي** - الرد على التعليقات تلقائياً
✅ **إدارة الرسائل المباشرة** - أدر رسائلك بسهولة
✅ **تقارير النمو** - قياس نموك ونجاحك
✅ **نظام المستخدمين** - تسجيل وتسجيل دخول آمن
✅ **نظام الدفع** - خطط مدفوعة متعددة
✅ **لوحة Admin** - إدارة شاملة للنظام

---

## 🛠️ التقنيات المستخدمة

### Frontend
- **React** - مكتبة واجهات المستخدم
- **Axios** - إدارة الطلبات
- **Redux** - إدارة الحالة
- **Tailwind CSS** - تصميم حديث

### Backend
- **Node.js** - بيئة التشغيل
- **Express** - إطار العمل
- **MongoDB** - قاعدة البيانات
- **JWT** - المصادقة الآمنة

### الخدمات الخارجية
- **Instagram Graph API** - ربط Instagram
- **Stripe** - معالجة الدفع

## 🚀 البدء السريع

### المتطلبات
- Node.js v14+
- MongoDB
- npm أو yarn

### التثبيت

1. **استنسخ المستودع:**
```bash
git clone https://github.com/kaldksa/VipTopSA.git
cd VipTopSA
```

2. **ثبت الاعتماديات:**
```bash
npm install
cd client && npm install && cd ..
```

3. **أنشئ ملف .env:**
```bash
cp .env.example .env
```

4. **أضف معلومات Instagram API:**
```
INSTAGRAM_APP_ID=your_app_id
INSTAGRAM_APP_SECRET=your_app_secret
INSTAGRAM_ACCESS_TOKEN=your_access_token
```

5. **شغل المشروع:**
```bash
npm run dev
```
